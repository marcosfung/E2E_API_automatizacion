Automatización API PetStore con Karate

Descripción

Se desarrolló una automatización sobre la API PetStore utilizando Karate Framework, cubriendo las operaciones CRUD solicitadas para la gestión de usuarios.

El flujo automatizado incluye:

- Creación de usuario
- Consulta de usuario creado
- Actualización de datos
- Validación de cambios realizados
- Eliminación de usuario
- Verificación de eliminación

## Herramientas utilizadas

- Java 17
- Maven 3.9
- Karate Framework 1.4.1
- Visual Studio Code
- GitHub

## Ejecución

Ubicarse en la raíz del proyecto y ejecutar:
mvn test
